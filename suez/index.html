Suez Eau France

Service A2I · Quint-Fonsegrives, Toulouse
10/2023 → 12/2023

Contexte

Suez Eau France gère un parc d'environ 160 télétransmetteurs Pérax (P400) connectés via un réseau VPN interne ("réseau technique"), permettant la surveillance d'installations d'eau potable et d'assainissement. Suite à l'annonce de l'arrêt des réseaux 2G/3G par les opérateurs, Suez devait identifier précisément les équipements à remplacer pour passer en 4G.

Ma mission principale était d'automatiser la collecte des numéros de série et versions logicielles de chacun de ces télétransmetteurs, accessibles uniquement via leur interface web interne — sans aucun accès à Internet depuis les postes de travail du réseau technique.

Projets réalisés

Script Python — Collecte automatisée des télétransmetteurs Utilisé en production

Web scraping des interfaces HTTP des ~160 télétransmetteurs Pérax P400 via la bibliothèque requests et des expressions régulières. Le script extrait le numéro de série et la version logicielle de chaque appareil, gère les timeouts et erreurs réseau, et exporte les résultats dans un fichier CSV — directement réutilisable par les équipes pour planifier les remplacements matériels.

Automatisation Excel — Dédoublonnage TOPKAPI Manuel en fallback

Recherche et tri automatique des libellés en doublon dans le superviseur TOPKAPI (outil de gestion des télétransmetteurs). Tentative d'automatisation via formules Excel et VBA ; blocage non résolu, solution manuelle adoptée en 20 minutes.

Contraintes techniques

L'ensemble du développement s'est fait sur un poste et une VM sans accès à Internet, connectés uniquement au réseau technique isolé de Suez. Chaque bibliothèque Python devait être transférée manuellement depuis un poste externe, puis installée localement — ce qui a nécessité de contourner les limitations de pip et de gérer les dépendances à la main.

Stack technique

Python Web scraping CSV requests Expressions régulières Excel / VBA HTTP Réseau VPN
