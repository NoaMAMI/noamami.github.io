Résidence hôtelière — VoIP & VOD sur IP

Lycée de l'Espace (Pierre-Paul Riquet) · Projet VM / OS Unix

Contexte

Un exploitant de résidence hôtelière souhaite moderniser son installation en déployant trois services : une téléphonie sur IP pour les chambres et les services internes, un système de vidéo à la demande accessible depuis un téléviseur Android dans chaque chambre, et un accès internet WiFi pour les clients.

J'ai conçu et déployé l'intégralité de cette infrastructure en solo, sur machines virtuelles VirtualBox, en partant de zéro : installation des OS Linux, configuration des services, câblage réseau virtuel, tests de bout en bout.

Architecture déployée

1

Infrastructure réseau

Conception du plan d'adressage IP, calcul des sous-réseaux, déploiement d'un switch et configuration des interfaces réseau des VMs (mode bridge vers le réseau physique de la salle).

2

Serveur VoIP — Asterisk / Trixbox

VM Linux (Trixbox, image ISO) hébergeant un IPBX Asterisk. Création des extensions SIP : poste 201 (Réception) et poste 202 (Chambre 1). Validation des appels entre softphones.

3

Clients SIP

microSIP sur PC Windows (poste Réception) et Linphone sur smartphone Android (poste Chambre). Enregistrement SIP, test d'appel bidirectionnel et messagerie vocale sécurisée (bonus).

4

Serveur VOD — VLC streaming

VM Debian 8 installée et configurée manuellement. Service VLC configuré avec deux flux de streaming (StarWars, FastAndFurious) définis dans une liste VLM. Transfert des fichiers vidéo par SCP.

5

Clients VOD

Configuration du client VLC sur les postes clients (PC Windows / Raspberry Pi Android TV) pour se connecter aux flux et visionner les contenus à la demande depuis le téléviseur de la chambre.

Stack technique

Linux Debian Asterisk / Trixbox VLC (streaming VLM) Protocole SIP / RTP VirtualBox TCP/IP · sous-réseaux microSIP Linphone SSH · SCP sudo · droits Unix APT (gestionnaire de paquets) Raspberry Pi · Android TV WiFi · routeur

Travail réalisé

  • Installation et configuration de deux VMs Linux sous VirtualBox (Debian 8 pour VOD, Trixbox pour VoIP) en partant de l'image ISO
  • Administration système Unix : gestion des comptes root/user, configuration de sudo, droits sur les fichiers, gestion des paquets avec APT
  • Configuration réseau des VMs en mode bridge, plan d'adressage IP et calcul des sous-réseaux de la résidence
  • Déploiement du serveur VoIP Asterisk (Trixbox) : création des extensions SIP, configuration des comptes Réception (201) et Chambre (202)
  • Installation et enregistrement des softphones (microSIP sur PC, Linphone sur smartphone) ; validation des appels et messagerie vocale sécurisée
  • Transfert des fichiers vidéo par SCP, configuration du service de streaming VLC avec liste VLM multi-flux
  • Tests de recette de l'installation (validation bout en bout avec le client)
